Current Version: 1.2.5

Repository addon

Instructions for adding this repo:

  • Go to the Kodi file manager.
  • Click on "Add source"
  • The path for the source is https://puppyrambo18.github.io/repo/ (Give it the name "Rambo's Repo").
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location, select "Rambo's Repo", and install repository.rambo-1.2.5.zip
  • Go back to Addons install, but this time, select "Install from repository"
  • Select the "Rambo's Repo"

Enjoy!
